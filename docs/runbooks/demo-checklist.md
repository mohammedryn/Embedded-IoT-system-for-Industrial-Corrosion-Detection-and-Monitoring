# Demo Checklist (Foundation)

## Pre-Demo

- [ ] `make bootstrap` completed.
- [ ] `make smoke-c00` passed.
- [ ] Config files present in `config/`.
- [ ] Logging path writable (`data/logs`).

## During Demo

- [ ] Check for fresh log entries in `data/logs/edge.log`.
- [ ] Verify timestamps update correctly.

## Post-Demo

- [ ] Archive logs under session directory.
- [ ] Record issues and fixes in runbook notes.
